# 🧠 Mood Swings Predictor

A Streamlit-based web application that predicts the likelihood of mood swings during the **menstruation** and **menopause** phases based on user-provided symptoms.

## Features

* 🩸 Menstruation Mood Swings Prediction
* 🌙 Menopause Mood Swings Prediction
* 📝 User Feedback Form
* 💡 Personalized health suggestions based on the prediction

## Technologies Used

* Python
* Streamlit
* Pandas

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/MoodSwingsPredictor.git
```

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
